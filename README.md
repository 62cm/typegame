# typegame

Horror typing survival game.

## Play

https://62cm.github.io/typegame/

- **卡通版（默认）** `index.html`
- **真人显示屏版** `monitor.html`（显示器旁烛光，鬼越近烛越暗；仅经典/剧情）
- **感冒模式** `cold.html`（手机全键盘，咳嗽溅唾液打学术报告）

右下角可切换版本。

## Modes

- **经典打字**：涂黑字 / 打叉字（该位必须填字，但不能填原字）/ 输入陷阱（红键封锁）
- **剧情模式**：鬼讲述生平、喜恶与人物；后续必须原样打出她说过的内容
- **九宫模式**（仅卡通版）：照常打字；右侧表情参考板（英文）。对照鬼脸按对应格
- **感冒模式**：视线方向抛物线喷射（无准心）；咳 3s / 喷嚏 10s（青黄浓鼻涕）；憋气 3s 大散射；石膏擦手机重置；开局空手机练习

## Rules

- **必须对上每一个字才能过关**（已取消凑字数）
- 涂黑字：打出前一个字后才显示
- 打叉字：这里要填充一个字，但不能是被划掉的原字；填原字招鬼，填别的字覆盖即可
- 二周目：通关后，每打对一字从文末涂黑一字
